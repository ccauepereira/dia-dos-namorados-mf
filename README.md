# Portal do Amor 💖

Um site interativo e romântico feito como presente especial de Dia dos Namorados para Maria Fernanda.

O projeto reúne memórias, fotos, carta, quiz, raspadinhas, música e uma surpresa final em formato de experiência digital personalizada.

## 🌐 Acesse o projeto

🔗 https://ccauepereira.github.io/dia-dos-namorados-mf/

## ✨ Funcionalidades

* Tela inicial com visual romântico
* Contador especial
* Linha do tempo com momentos importantes
* Galeria de fotos do casal
* Carta personalizada
* Quiz romântico
* Raspadinhas com prêmios especiais
* Música de fundo
* Surpresa final
* Seção Premium Finale com Barbie Fernanda™
* Checkout fictício romântico com XereCard™

## 🛠️ Tecnologias usadas

## 🚀 Como rodar localmente

Clone o repositório:

```bash
git clone https://github.com/ccauepereira/dia-dos-namorados-mf.git
```

Entre na pasta:

```bash
cd dia-dos-namorados-mf
```

Inicie um servidor local:

```bash
python3 -m http.server 8000
```

Depois abra no navegador:

```text
http://localhost:8000
```
